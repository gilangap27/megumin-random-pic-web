HOW TO USE:

  1. To get a random megumin image, you can press the random button
  2. Then system will fetch megumin image url with API
  3. Setelah mendapatkan url maka sistem akan mengganti alamat url gambar ke url random gambar yang baru
  4. After getting the url, the system will change the image url address to a new random image url
  5. Please don't spam random buttons because it can cause errors
